# Contributing to AspNetConventions

First of all — thank your interest in contributing to **AspNetConventions**. That already means a lot.

This project is built around one simple idea: **make ASP.NET Core applications more consistent through conventions.** If you want to help improve that mission, you’re in the right place.

## Ways You Can Contribute

There are several ways to contribute:

- [ ] Write or improve tests
- [ ] Report bugs or unexpected behavior
- [ ] Review and resolve open issues
- [ ] Improve documentation and examples
- [ ] Optimize performance
- [ ] Suggest or implement new features
- [ ] Share ideas that improve this library


## How to Get Started
Check out the ["contributing to a project"](https://docs.github.com/en/get-started/exploring-projects-on-github/contributing-to-a-project) guide.
